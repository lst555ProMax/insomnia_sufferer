# 人缘何学会失眠

前一晚心流满满，做出美妙奇丽的设想，一觉醒来突然发现自己的想法无比幼稚

又或者，一步步推想自己要做的事情，醒来之后却根本没有勇气去做

这种事情常常发生

INTJ最习惯的就是冥想，这种习惯到了夜里，就变成了空想

我常常把自己称作空想主义者

空想主义和理想主义有多大的区别呢

绝大多数人的一生，我们可以看做是理想主义 ->  现实主义的一生

我们下一个定义名叫清醒人

清醒人生于人的觉醒，死于人的入睡，他们的寿命只有十几个小时

清醒人的一生恰恰相反，他们生于现实，死于理想

生的匆匆忙忙，一天有太多事情要做，各种工作的、日常的、人际的、人情的、突发的、自发的…

当人躺倒在床上执行例行的睡眠之时，清醒人的生命达到高潮，他们为了避免死于现实，开始了一轮又一轮的狂想

他们成为了一名勇敢的表白者

他们成为了一位卓越的工程师

他们成为了一位充满创意的创业者

他们成为了一位小众的歌手

他们成为了一名表达概念的艺术家

他们成为了画家，有的时候是莫奈，有的时候是梵高，是塞尚，是蒙克，是达芬奇

他们有时候会成为无比敏感的人，任何一件小事都会在心里泛起涟漪

有时候是极度自卑的人，过往的错误会突然让他们满脸羞愧

有时候是一位冷漠的记者，只会记录所有意识的流动

他们还可以是小说家，是探险家，是历史学家，是政治家，是电影导演，是魔术师…

他们发现，只要不停地狂想，就真的可以一直活下去

他们是优秀的rule breaker，原来2点的时候还可以保持清醒，3点，4点...

直到一个朦胧的时刻，人终于战胜了失眠，进入了梦乡，狂想在这一刻随着清醒人的生命消逝了

梦是另一个魔幻的世界，美好与险恶交汇

梦醒之后，一个新的清醒人诞生，梦不会连接两个清醒人的思想

梦会让人失忆，今天是星期一

人们忘记了几个小时前的决心和誓言，又开始了忙忙碌碌

"你怎么了？"

"没什么，晚上失眠了"

# 失眠的文学重构 - AI评价对比网页

这是一个用于展示原始文章和两个AI模型（千问Deep Research和Gemini Deep Research）评价对比的静态网页项目。

## 技术栈

- React 18
- TypeScript
- Vite
- SCSS Modules
- React Router
- React Markdown (用于渲染Markdown内容)
- remark-gfm (支持GitHub风格的Markdown)


## 安装依赖

```bash
npm install
```

## 开发运行

```bash
npm run dev
```

## 构建生产版本

```bash
npm run build
```
